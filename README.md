## 一键配置

```bash
$ curl https://gitee.com/xnng/bash/raw/master/install.sh | sh
```

>用管理员身份打开 `Git Bash`，然后再运行命令，期间会开两个窗口，需要手动复制下文件，这个过程是在安装字体，命令无法操作，[动图演示](https://xnngs.oss-cn-shanghai.aliyuncs.com/img/bash.gif)
